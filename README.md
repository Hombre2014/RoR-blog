## Usage

Run the following commands:</br>
`git clone https://github.com/Hombre2014/RoR-blog`</br>
`cd Ror-blog`</br>
`bundle install`</br>
`rails s`</br>
Open a browser and go to: `https://localhost:3000` to view the app.
